Altstore安装教程

l1nsn0w 2020-11-02 Altstore

# 准备工作：

1.AltStore工具只能在Windows 10系统下使用，其他Windows版本不支持。

2.安装之前：

  1. 直接从Apple (opens new window)（而不是Microsoft Store）下载最新版本的iTunes 。

  2. 直接从Apple (opens new window)（而不是Microsoft Store）下载最新版本的iCloud 。

# 安装开始

# 一、下载安装AltStore工具

官网下载 (opens new window)

下载好的是一个压缩文件【altinstaller】，解压开之后双击【setup】安装。

安装好之后就会看到【AltServer.exe】文件。

如果被Windows防火墙阻挡，请将专用网络和公用网络的勾都打上，然后点击允许访问。

# 2.使用altstore

将iPhone通过数据线连接iTunes，确认是否连接成功，若连接成功，即可进行下一步。

根据提示输入AltServer.exe文件路径，比如：E:\altinstaller\AltServer.exe（如果不需要这步请无视），点确定。

点击Utility to Install选项，先将已生成的AltStore下载地址复制起来备用，然后选择Custom IPA Link，将AltStore下载地址的短网址复制到对话框中（因为对话框内只支持最多55个字符）。

如跳出以下窗口，点击【确定】继续

此时，我们可能仍然无法在Windows桌面找到AltServer 工具，我们可以通过左下角的Windows系统放大镜功能，输入【alt】来寻找。

# 3.安装altstore

在运行了AltServer工具之后，点击Windows桌面右下角的菱形图案，选择Install AltStore，并点击你的设备名称，此时会跳出要求你输入Apple ID和密码，如果担心安全问题，请提前注册新的Apple ID。

（如果没看到你的设备名称，那就是之前连接iTunes不成功，或被Windows防火墙阻挡。）

4.接下来会跳出一个询问窗口，询问你是否要将AltStore工具安装到iOS设备里，如果没看到以下窗口，请确认使用的Apple ID和密码是否有误，或之前复制AltStore的下载地址是否有误。

接下来就会看见Installing AltStore to+你的设备名称 … ，代表AltStore正在签名安装中。

**说明：**整个安装AltStore进的过程有长有短，在iPhone桌面出现AltStore APP之后，安装过程才算完成，此时先别动，到设置里的设备管理中信任一下，如无法信任，请重新安装。

# 常见问题及避坑指南

# altstore不为我安装

我按照说明进行操作，为什么AltStore仍不为我安装？发生这种情况可能有多种原因，但据报道这些解决方案有所帮助：

  • 将其连接到计算机后，您是否在计算机和电话上都说了“信任”？您可以通过打开iTunes进行检查，然后查看是否弹出对话框询问您是否要信任手机。
  • 尝试使用另一个Apple ID。如有必要，您可以免费创建 (opens new window)专门用于AltStore的新Apple ID (opens new window)。
  • （仅Windows）确保使用AltServer时iTunes和/或iCloud正在运行。
  • （仅限Windows）右键单击AltServer，然后选择“以管理员身份运行”。
  • （仅Windows）您是否从Microsoft Store安装了iTunes或iCloud？如果是这样，则需要卸载它们并直接从Apple下载最新版本。

# AltStore永久冻结/永久登录。

发生这种情况可能有多种原因。如果遇到此问题，请检查以下内容：

  • （仅Windows）您的Windows防火墙可能正在阻止与AltServer的传入网络连接。您必须在防火墙设置中为AltServer启用网络访问，才能从AltStore接收刷新的应用程序。
  • （仅Windows）您是否从Microsoft Store安装了iTunes或iCloud？如果是这样，则需要卸载它们并直接从Apple下载最新版本。
  • （仅限Windows）您可能需要确保iTunes和iCloud也在计算机上运行。
  • 您在公共/工作/学校WiFi上吗？如果是这样，您的WiFi可能会阻止设备相互发现。尝试连接到手机的热点，然后重试。
  • 将其连接到计算机后，您是否在计算机和电话上都说了“信任”？您可以通过打开iTunes进行检查，然后查看是否弹出对话框询问您是否要信任手机。
  • 最后，尝试将手机插入计算机。这将解决几乎所有的连接问题，但将意味着AltStore很可能将无法在后台通过WiFi为您刷新应用程序。

# 建议使用AltStore的方式是什么？

  • 将AltServer设置为在打开/登录计算机时自动启动。
  • 不使用计算机时将其置于睡眠模式，并在可能的情况下插入计算机（尤其是在晚上，因为插入电话且不执行任何操作时，iOS在后台唤醒应用程序的频率会更高）。
  • 每天/每隔几天打开一次AltStore，以便iOS学会确定其优先级，并使其尽可能在后台刷新应用程序。
  • 不要紧张 AltStore旨在在一周中多次检查。随身携带笔记本电脑，关闭台式机。只需确保它每隔一段时间就处于活动状态，以便AltStore可以刷新🙂

# 如何通过AltStore侧载其他应用程序（.ipa文件）？

在“我的应用”标签中，点击左上角的“ +”按钮，然后选择您要侧加载的.ipa文件。

# 我是否需要担心“吊销”？

否。其他替代应用商店使用“企业证书”来允许人们安装应用。但是，Apple明确禁止这样做，导致他们积极关闭或“撤消”了其他应用商店使用的这些企业证书。这会导致应用程序每隔一段时间停止打开一次，直到获得新的企业证书为止。

由于AltStore不使用企业证书，因此不必担心Apple会通过撤销单个证书来关闭AltStore（因为从技术上来说，每个人现在都有自己的签名证书）。

# 为什么需要我的Apple ID？

Apple允许具有Apple ID的任何人将自己构建的应用程序安装到设备上进行测试。AltStore使用您的Apple ID代表您与Apple的服务器进行通信，并执行必要的步骤来准备将帐户安装到设备上的帐户。
